# ValenciaGameJam2026 - Heiwa: Mask Spirits
https://globalgamejam.org/games/2026/heiwa-5

## Objetivo de Juego
¡Consigue los 4 fragmentos de mascara usando tus habilidades para controlar el tiempo de los objetos!

## Controles
### Teclado
Movimiento Izquierda: A
Movimiento Derecha: D
Saltar: Barra Espaciadora/Space
Apuntar: Ratón/Mouse
Disparar: F
Rebobinar: R (apuntando a un objeto)
